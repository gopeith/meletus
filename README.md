# Meletus

Tento projekt soustřeďuje znalosti a skripty pro využítí AI při vedení internetových diskusí.

Popis souborů:

 * rules.txt - soupis základních pravidel vedení diskuse. Pozor, soubor neobsahuje konkrétní strategie a rétorické techniky, jen základni etické principy.
 * strategy.txt - soupis pravidel, jak je možné diskusi vést aby byly dodrženy principy obsažené v souboru rules.txt.
